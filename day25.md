# Day 25 Javascript 
# What are the different way To create objects in js ?
# 1.object literal
```jsx
const person={
    name:'kaif',
    id:'2',
    age:'21'
}
console.log(person)

```
# 2 using Constructor 
#### with the help of this we can create  empty objects and add properties later
```jsx
const person=new Object()
person.name="kaif"
person.age="21"
console.log(person)
```
